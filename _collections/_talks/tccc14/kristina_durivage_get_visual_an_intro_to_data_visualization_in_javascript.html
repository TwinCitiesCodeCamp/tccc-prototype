D3.js is a javascript library used to make visualizations off of data. It goes very well with HTML, CSS and SVG to make dynamic and interesting graphics from static documents or API calls. This talk will go over making a simple data visualization. Also covers the Miso Dataset library for data import and and overview of other useful tools and places for help. This is code-centered talk.
