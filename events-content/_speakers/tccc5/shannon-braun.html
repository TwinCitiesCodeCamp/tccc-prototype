--- speakerId: ShannonBraun name: Shannon Braun image: speakerUrl: twitter: github: event: tccc5 --- Shannon Braun is an independent consultant and founder of Sysknowlogy, which provides consulting and development services using Microsoft technologies. Shannon focuses on assisting corporations adopt Microsoft technologies and has helped companies solve business problems with Microsoft .NET, BizTalk, Commerce Server, SharePoint, Windows Presentation Foundation (WPF) and Web Services.

Shannon has served as a member of the .NET Partner Advisory Committee, Microsoft Architecture Partner Board, Microsoft Architecture Advisory Board and is currently a member of the Visual Studio Partner Advisory Board. Shannon was awarded a Microsoft Solution Architect MVP for 2006 and 2007 and a Microsoft Client Application Development MVP for 2008.
