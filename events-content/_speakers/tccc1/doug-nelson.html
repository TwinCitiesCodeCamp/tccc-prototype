--- speakerId: DougNelson name: Doug Nelson image: speakerUrl: twitter: github: event: tccc1 --- Doug Nelson is a principal consultant with ILM. He provides expert application development services for several large clients with a primary focus in the forest products industry and industrial health and safety. These applications developed are based on a SQL Server backend database with a web based front-end. He has been doing web development since 1996, transitioning from CGI applications to classic ASP and on to ASP.NET. In addition to the web development, these applications also support WinForms and Pocket PCs. He has extensive experience implementing truly distributed applications involving windows services and message queuing.
